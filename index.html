Scratch

A Library by Mike Barkmin for the Processing programming environment.
Last update, 10/25/2018.

Provides classes to match the functionality of scratch

For more detailed information visit the GitHub Project Page.

Download

Download Scratch version 1.3.0 (4) in .zip format.

Installation

Unzip and put the extracted scratch folder into the libraries folder of your Processing sketches. Reference and examples are included in the scratch folder.

Keywords. scratch

Reference. Have a look at the javadoc reference here. A copy of the reference is included in the .zip as well.

Source. The source code of Scratch is available at GitHub, and its repository can be browsed here.
